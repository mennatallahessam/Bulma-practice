# Bulma Practice

This repository is dedicated to practicing web development using the [Bulma CSS Framework](https://bulma.io/).

## Current Project: Menu Recreation
A practice exercise to recreate a classic application-style menu using Bulma components.

### Goals:
- [x] Set up Bulma via CDN
- [x] Implement a responsive Navbar
- [x] Use Bulma's `box` and `section` components
- [x] Recreate Todo practice page (`no-framework/index.html`)
- [ ] Practice layout customization
- [ ] Experiment with Bulma modifiers (colors, sizes, etc.)
